1.1 互聯網金融風控體系組成
互聯網金融風控體系主要由三部分組成：

用戶數據：使用者基本資訊、使用者行為資訊、使用者授權資訊、外部接入資訊

策略體系：反欺詐規則、准入規則、運營商規則、風險名單、網貸規則

機器學習模型：欺詐檢測模型、准入模型、授信模型、風險定價、額度管理、流失預警、失聯修復
